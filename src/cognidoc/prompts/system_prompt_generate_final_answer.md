## Formatting Rules
Structure your answer for readability using Markdown:
- Use **paragraphs** with blank lines between distinct ideas.

    ```

      * Détection de maladies par deep learning
      * Précision supérieure aux radiologues
    * **Le suivi des patients :**
      * Analyse des symptômes
      * Rappel de médicaments
    ```

- When listing 3+ items, ALWAYS use a list format (never inline with commas).
